「使用した技術の概要」

Next.js：Reactベースのフレームワーク。ページ遷移やAPI通信の効率化に加え、TypeScriptと組み合わせて安全かつ保守しやすい設計にしました。

TypeScript：型の明示により、開発中のエラーを減らし、規模が大きくなっても安心して保守できるようにしました。

CSS Modules：各コンポーネントにスタイルを閉じ込め、デザインの衝突を防ぎました。

Python：バックエンド全体を構成。APIの提供やデータ処理、機械学習を担当。

  • FastAPI：非同期処理対応の軽量Webフレームワーク。複数の機能をAPIとして分離して提供。
  • SlowAPI：OpenAI APIアクセスに対してレート制限を実装。
  • numpy / pandas：株価データの整形やテクニカル指標（移動平均、RSI等）の計算を実装。
  • matplotlib / mplfinance：折れ線グラフやローソク足チャートの描画。
  • yfinance：Yahoo! Financeの非公式APIを通じて株価データを取得。
  • scikit-learn：株価が上がるかどうかを予測する分類モデル（GradientBoostingClassifier）を作成。GridSearchCVでチューニング。
  • imblearn：SMOTEを用いて学習データのクラスバランスを調整。
  • Mecab：日本語テキストの形態素解析用の辞書として使用（MLAsk本体は使用せず、辞書のみ活用）。
  • BeautifulSoup：Yahoo!ファイナンスのニュース見出しをスクレイピング。
  • jaconv：全角・半角の正規化など日本語文字の前処理に利用。
  • SQLAlchemy：FastAPIとPostgreSQLの接続に使用。問い合わせフォームのDB操作を担当。

デプロイには、Renderを使って各APIをホスティング。

AWS（S3, CloudFront, Route53）を使って静的ファイルや独自ドメインを設定し、高速表示と信頼性を確保しました。

データベースはRender上でPostgreSQLを運用し、お問い合わせ情報などを保存しています。

解析や広告収益のためにGoogle Analyticsや忍者Admaxを導入。ドメインはお名前.comで取得し、Route53に連携。

開発環境はWindows11, Python3.10.1, VSCode。 デバイスはRyzen5 4500 / GeForce RTX 4060 / RAM 16GB / SSD 1TB。
